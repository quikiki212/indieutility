Frequently asked questions

Which browsers support glassmorphism?

Modern Chrome, Edge, Safari, and Firefox support backdrop filtering. The generated CSS includes the WebKit-prefixed property for broader compatibility.

Does this upload my design?

No. The editor runs entirely in your browser and does not send your settings anywhere.

Can I use the CSS commercially?

Yes. Copy, modify, and use the generated styles in personal or commercial projects.
