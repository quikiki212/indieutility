Frequently asked questions

Is my SQL uploaded?

No. Formatting happens entirely in your browser.

Does the formatter execute queries?

Never. It only restructures text and does not connect to a database.

Which SQL dialects are supported?

The formatter handles common Standard SQL, PostgreSQL, MySQL, SQLite, and SQL Server syntax without changing query semantics.
